# Xmind

  • 我的写作工具和习惯
  • 重视快捷键，重视工具
  • 一定要重视工具的使用，对于常用的工具，要观察好自己的使用习惯，积累自己使用的定制方案

# Xmind快捷键『转载』

# 01.九键速成版

  • 记住 9 个快捷键，XMind 就算学会了。
1、新建与保存
（1）新建空白图：Ctrl+Shift+N
（2）保存：Ctrl+S


2、添加主题
（1）『并列』主题：Enter
（2）『子』主题：Tab
（3）父主题：Ctrl+Enter


3、标注整理
（1）『备注』：Fn+F4
（2）标签：Fn+F3
（3）外框：Ctrl+B
（4）概要：Ctrl+】

# 02.自学精进版

  • 如果想精进 XMind，继续学习以下内容。

# 2.1.文件

新建：Ctrl+N
新建空白图：Ctrl+Shift+N
打开：Ctrl+O
关闭：Ctrl+W
关闭全部：Ctrl+Shift+W
保存新的版本：Ctrl+S
保存全部：Ctrl+Shift+S
打印：Ctrl+P

# 2.2.编辑


撤销：Ctrl+Z
重做：Ctrl+Y
剪切：Ctrl+X
拷贝：Ctrl+C
粘贴：Ctrl+V
删除：Delete
回到中心主题：Ctrl+Home
全选：Ctrl+A
选中同级主题：Ctrl+Shift+A
选中子主题：Ctrl+Alt+A
查找/替换：Ctrl+F

# 2.3.查看

开始头脑风暴：F8（此功能需购买 XMind 8 Pro）
实际大小：Ctrl+0
缩小：Ctrl+- 放大：Ctrl++
上钻：Shift+F6
下钻：F6

# 2.4.插入

主题：Enter
子主题：Tab
主题（之前）：Shift+Enter
父主题：Ctrl+Enter
标注：Alt+Enter
联系：Ctrl+L
外框：Ctrl+B
概要：Ctrl+】
图片来自文件：Ctrl+I
备注：Fn+F4
标签：Fn+F3
超链接：Ctrl+H
新画布：Ctrl+T
从主题创建新画布：Ctrl+Alt+T
从模板创建新画布：Ctrl+Alt+N
